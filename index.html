About Me

Perkenalkan, nama saya Muhammad Maulana. Saya seorang Beginner Front End Developer yang berasal dari Palembang, Sumatra Selatan, tepatnya di daerah Ilir Timur II.

Ketertarikan saya pada dunia Front End dimulai setelah lulus Sekolah Menengah Atas. Sejak kecil, saya sudah terbiasa bermain komputer, yang kemudian menumbuhkan minat saya pada bidang Teknologi Informasi. Karena itu, saya memilih jurusan Teknik Informatika setelah menyelesaikan pendidikan SMA.

Rasa penasaran terhadap ilmu pemrograman membawa saya menemukan saluran YouTube Web Programming UNPAS, yang menjadi titik awal perjalanan saya di dunia Front End Development.

Dan akhirnya, saya melanjutkan kuliah di Universitas Sriwijaya, Teknik Informatika sampai saat ini.

Melalui Portofolio ini, saya ingin berbagi hasil-hasil proyek yang telah saya kerjakan. Proyek-proyek ini akan mencerminkan perjalanan saya dalam belajar dan berkembang sebagai seorang programmer.
